Veekls public API client WordPress plugin.

== Description ==

This plugin provides easy integration with the Veekls public API just using your API Key.

== Installation ==

1. Search for 'Veekls' on the WordPress Plugin Directory.
1. Activate the plugin through the 'Plugins' menu in your WordPress admin.
1. Configure your API Key on the 'Veekls API' menu in your WordPress admin.
1. Place `<?php do_action('veekls_api_client_hook'); ?>` in your templates

== Frequently Asked Questions ==

= How do I get my API Key? =

You must ask your Veekls agent for it. Do not share it!

= Are there any limits on using the API? =

There are none at the moment.
